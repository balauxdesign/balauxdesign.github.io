Target design system

Target Design Systems is a comprehensive set of guidelines, components, and tools that ensure a consistent and scalable design experience across Target’s digital products. It provides a unified visual language, accessibility standards, and reusable UI components that help designers and developers create intuitive, user-friendly interfaces. The system focuses on efficiency, brand alignment, and seamless customer interactions, supporting both web and mobile platforms while maintaining Target’s distinct identity.

As a Senior Product Designer working on the Target Design System, I played a key role to shape and evolve the design language that powers Target’s digital experiences. I collaborate closely with designers, engineers, and product teams to create scalable, accessible, and visually cohesive UI components that enhance usability across web and mobile platforms.

Remote PC

RemotePC is a secure and reliable remote access solution that allows users to connect to their computers from anywhere, anytime.
